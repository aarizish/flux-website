# Flux Website

This is the **official documentation and download website** for the [Flux App Inventor Extension](https://github.com/aarizish/flux-extension).
Built with **React** and **Vite**, it provides an elegant and fast interface for exploring documentation, viewing examples, and downloading the latest `.aix` builds of **Flux**.

## Overview

**Flux Website** serves as:

* A **Documentation Hub** for guides, API references, and usage examples.
* A **Download Portal** for the latest `.aix` releases.
* A **Showcase** of Flux’s capabilities and roadmap.

Built for clarity, speed, and accessibility, powered by **React**, **Vite**, and **Markdown-based documentation**.

## Tech Stack

| Tool                                                  | Purpose                     |
| ----------------------------------------------------- | --------------------------- |
| [React](https://react.dev/)                           | UI library                  |
| [Vite](https://vite.dev/)                             | Fast bundler and dev server |
| [React Router](https://reactrouter.com/)              | Client-side routing         |
| [Markdown / MDX](https://mdxjs.com/)                  | Documentation content       |

## Project Setup

### Clone the Repository

```bash
git clone https://github.com/aarizish/flux-website.git
cd flux-website
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Your site will be available at [http://localhost:5173](http://localhost:5173).

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
